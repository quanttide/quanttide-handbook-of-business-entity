# AGENTS.md

本仓库是量潮科技工作手册系统中的公司手册。

## 工作流程

每次生成或修改文档时，必须遵循以下顺序：

第一步：阅读格式规范。打开文档格式标准链接，理解删、简、少、一的原则。

第二步：生成文档内容。根据需求创建或修改文档，避免过度格式化。

第三步：对照规范优化。检查加粗、列表、标题层级、引号等格式元素，按CONTRIBUTING中的优化方法调整。

第四步：验证并提交。确认格式符合规范后提交推送。
